概要

Claude Code って何ができるの？

ターミナル上で Claude AI と直接対話し、大規模なコードベースを理解・編集・自動化できるエージェント型ツールです。

📂

コードベースを理解する

プロジェクト全体のファイルを読み込み、構造・依存関係・バグの原因を把握します。

✏️

コードを書いて編集する

指示するだけで新機能の実装、リファクタリング、テストコードの作成を行います。

⚡

コマンドを自動実行する

ビルド・テスト・Git 操作など、ターミナルコマンドを自律的に実行・修正します。

仕組み

どうやって動いているの？

ユーザーがチャットで指示するだけで、Claude Code が自律的に以下のサイクルを回します。

Step 1
💬

指示を出す

自然な日本語や英語で「このバグを直して」と伝えるだけ

Step 2
🔍

コードを読む

関連ファイルを自動で探索し、問題箇所を特定する

Step 3
🛠️

編集・実行

ファイルを書き換え、コマンドを実行してテストする

Step 4
✅

確認・完了

変更内容を報告し、承認があれば完了する

実際の使い方

ターミナルでの実際の操作

こんなふうにやりとりするだけで、Claude Code がコードを自動で変更してくれます。

~/my-project — claude
$ claude
# Claude Code が起動します

FAQ

クリックして回答を展開できます。

Claude Code は無料で使えますか？
Claude Code は Anthropic の API を使用するため、API の利用料金が発生します。Claude.ai の Pro プランや API キーが必要です。利用量に応じた従量課金制です。
日本語での指示に対応していますか？
はい、完全対応しています。日本語で「このバグを直して」「新しい API エンドポイントを追加して」と指示するだけで動作します。コード自体は英語のまま扱えます。
どのプログラミング言語に対応していますか？
Python、JavaScript/TypeScript、Go、Rust、Java、C/C++、Ruby、PHP など、主要な言語はすべて対応しています。また、HTML/CSS、SQL、Shell スクリプト、設定ファイル (YAML, JSON, TOML) なども扱えます。
コードの安全性は大丈夫ですか？
ファイルの変更やコマンドの実行前に確認を求めます。また、Git 管理下のプロジェクトであれば、変更をレビューして git revert で元に戻すことも簡単です。機密ファイルを .claudeignore に追記して除外することも可能です。
VS Code などの IDE と一緒に使えますか？
はい、ターミナルで動作するため、どの IDE と併用しても問題ありません。VS Code のターミナルで Claude Code を起動し、エディタで変更内容を確認するという使い方が一般的です。
